### BTree - Pesquisa Binária

```haskell
data LTree a = Leaf a | Fork (LTree a, LTree a) deriving (Show, Eq)

data BTree = Empty | Node (A,(BTree, BTree))

lookBTree :: Ord a => a -> BTree (a,b) -> Maybe a
lookBTree a Empty = Nothing
lookBTree a (Node((a',b'),(l,r)))
    | a == a' = Just b'
    | a < a' = lookBTree a l
    | a > a' = lookBTree a r
```
